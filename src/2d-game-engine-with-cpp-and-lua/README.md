# 2D Game Engine with C++ and Lua

## Introduction

This is my coursework for Pikuma's [2D Game Engine course](https://pikuma.com/courses/cpp-2d-game-engine-development).

## Differences

I've used C++23, C++ modules, and vcpkg for dependency management, including newer versions of the libraries. I've not intended to follow the course's cross-platform structure, mainly since I work on Windows anyway.
